# Known Exclusions (v1)

The following are intentionally excluded from this repository:

1. Raw-audio processing implementation details
2. Raw-audio diarization setup and operational notes from private environments
3. Real transcript content and diarization artifacts
4. Any personal/work identifiers and real-world relationship context
5. Production credentials, logs, and internal infrastructure references

These exclusions are mandatory to preserve privacy, reduce attack surface, and keep the public project architecture-focused.
