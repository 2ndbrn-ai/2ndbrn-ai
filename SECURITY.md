# Security Policy

## Scope
This repository is a public architecture/spec project.
It must never contain credentials, private transcripts, raw audio artifacts, or production-sensitive details.

## Reporting a Vulnerability
Please report security issues privately by emailing:

- security@2ndbrn.ai

Include:
- clear description of the issue
- affected file(s)/path(s)
- reproduction steps (if applicable)
- potential impact

Please do not open public issues for undisclosed vulnerabilities.

## Response Targets
We’ll acknowledge receipt as soon as we can and follow up with triage status.
