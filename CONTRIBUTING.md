# Contributing

Thanks for your interest in improving 2ndBrn.

## Project Scope
This repository is for **public-safe architecture and specification artifacts**.

### Allowed contributions
- architecture docs
- diagrams
- threat model/privacy model improvements
- clarification PRs
- reference scaffold proposals (sanitized)

### Not allowed
- private transcripts or transcript excerpts
- raw audio/video artifacts
- credentials/tokens/secrets
- production logs or private infrastructure details
- personal/work-identifying content
- PRs that include disallowed content may be closed without merging

## How to Contribute
1. Open an issue describing the proposal/problem.
2. Keep PRs small and focused.
3. Include rationale + tradeoffs.
4. Link related docs/sections updated.

## Pull Request Checklist
- [ ] No sensitive/private data included
- [ ] No secrets/tokens/keys
- [ ] No personal identifiers
- [ ] Scope remains architecture/spec (not private implementation)
- [ ] Docs updated for any behavior/terminology change
