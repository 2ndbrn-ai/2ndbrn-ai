# 2ndBrn.ai

2ndBrn.ai is a transcript-first, privacy-first personal knowledge operating system.
This repository is the public specification and reference scaffolding.

## Status
Early-stage. Spec-first.

## What’s here
- /docs – sanitized public architecture documentation
- /diagrams – architecture diagrams (public-safe only)
- /spec – spec documents and publish boundaries

## Start here
- Architecture: `docs/architecture.md`
- Pipeline overview: `docs/pipeline.md`
- Security reporting: `SECURITY.md` (`security@2ndbrn.ai`)
- Contribution guide: `CONTRIBUTING.md`

## Scope (v1)
- Transcript-first pipeline architecture fed by diarized transcript inputs
- Reference docs, interfaces, and operating model

## Non-goals
- Production deployment internals
- Raw-audio processing implementation details
- Hosted SaaS runtime
