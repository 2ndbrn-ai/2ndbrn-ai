# 2ndBrn

2ndBrn is a transcript-first, privacy-first personal knowledge operating system.
This repository is the public specification and reference scaffolding.

## Status
Early-stage. Spec-first.

## What’s here
- /docs – public documentation
- /diagrams – architecture diagrams
- /spec – spec documents and conventions

## What’s NOT here
No private transcripts, keys, logs, databases, or production integrations.
