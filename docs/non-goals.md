# Non-Goals (Sanitized Public Repo)

This repository is **not**:
- A dump of production configuration
- A corpus of personal transcripts
- A raw-audio processing toolkit
- A turnkey hosted SaaS product
- A replacement for security/privacy controls

## Explicit exclusions
- Raw-audio diarization implementation details from private environments
- Provider tokens, account wiring, private network maps
- Real names, biographies, or sensitive occupational context

## What success looks like
- Reproducible architecture understanding
- Clear safety/privacy boundaries
- Community critique on design decisions
- Clean separation between public spec and private operations
